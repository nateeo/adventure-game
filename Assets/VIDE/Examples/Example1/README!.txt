To use the JS example scene:

1. HAVE VIDE_Data, VIDE_Assign, and MiniJSON_VIDE inside a 'Plugins' folder in the project root
2. Uncomment JS_exampleUI.js and JS_demoPlayer.js
3. Make sure the scripts are correctly attached to the 'UIManager' and 'Player' game objects in the scene.

Refer to the Documentation for more information.
